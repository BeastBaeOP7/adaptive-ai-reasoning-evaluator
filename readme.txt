# 🧠 Adaptive AI Reasoning Evaluator

A machine learning system that predicts whether AI-generated reasoning is correct using structured features and text-based analysis.

---

## 🚀 Features

* Predicts correctness of AI reasoning (~90% accuracy)
* Uses TF-IDF for semantic understanding
* XGBoost model for high performance
* Adaptive memory system to learn from past mistakes
* Interactive CLI tool for testing

---

## 🛠 Tech Stack

* Python
* Scikit-learn
* XGBoost
* Pandas

---

## 📊 Results

* Accuracy: ~90%
* Balanced precision and recall
* Improved from ~85% → ~90% using TF-IDF + XGBoost

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python src/train.py
python src/app.py
```

---

## 💡 Key Insight

Longer or more complex reasoning does not guarantee correctness — the model learns patterns that indicate failure.
